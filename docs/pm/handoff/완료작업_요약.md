- **완료일**: 2026-02-10
- **담당**: Backend Agent + Mobile Agent
- **내용**:
  - 백엔드: `PUT /api/users/password` API 구현

  - 새 비밀번호 정책 검증 (8자, 3가지 조합)
  - 실시간 비밀번호 강도 표시
  - 소셜 로그인 사용자 차단

### TASK-002: OAuth 소셜 로그인 (Google/Kakao)
- **완료일**: 2026-02-11
- **담당**: Backend Agent + Mobile Agent
- **내용**:
  - 백엔드: `POST /api/auth/social/login` API 구현 (Google, Kakao)
  - 모바일: Google/Kakao 로그인 버튼 연동
- **결과**:
  - Google OAuth UserInfo API 연동
  - Kakao OAuth UserInfo API 연동
  - 신규 사용자 자동 회원가입
  - 기존 사용자 로그인 + JWT 발급
  - Clean Architecture 구조 준수
  - Android/iOS 플랫폼 설정 완료
- **비고**: Apple Sign-In은 Apple Developer 계정 확보 후 추후 진행
