# PM 관리 문서

이 폴더는 PM이 관리하는 프로젝트 문서들이 저장됩니다.

---

## 📂 하위 폴더

### 📋 specs/ - 기능 명세서
새로운 기능 개발을 위한 상세 명세서

**포함 내용**:
- 비즈니스 요구사항
- 기술 요구사항
- 사용자 스토리
- 수용 기준 (Acceptance Criteria)
- 예상 개발 기간
- 우선순위

**파일명 예시**:
- `SPEC_티켓검증시스템_v1.0.md`
- `SPEC_알림시스템_v1.0.md`

---

### 🤝 handoff/ - 인수인계 문서
작업 완료 후 팀 간 인수인계 문서

**포함 내용**:
- 완료된 작업 요약
- 변경 사항 목록
- 테스트 결과
- 알려진 이슈
- 다음 단계

**파일명 예시**:
- `HANDOFF_2026-02-09_거래내역기능.md`
- `HANDOFF_2026-02-05_구매확정API.md`

---

### 📅 meetings/ - 회의록
정기 회의 및 중요 논의 기록

**포함 내용**:
- 참석자
- 안건
- 논의 내용
- 결정 사항
- 액션 아이템

**파일명 예시**:
- `2026-02-09_주간회의.md`
- `2026-02-01_스프린트리뷰.md`

---

### 🚀 release_notes/ - 릴리즈 노트
버전별 릴리즈 정보

**포함 내용**:
- 버전 정보
- 릴리즈 날짜
- 새로운 기능
- 버그 수정
- 알려진 이슈
- 업그레이드 가이드

**파일명 예시**:
- `RELEASE_v1.0.0_2026-03-01.md`
- `RELEASE_v1.1.0_2026-04-01.md`

---

## 📝 작성 가이드

각 하위 폴더에는 `_TEMPLATE.md` 파일이 있습니다. 새 문서 작성 시 템플릿을 복사하여 사용하세요.
